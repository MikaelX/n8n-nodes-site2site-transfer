# Images Directory

This directory contains screenshots and visual references for the Site to Site File Transfer node.

## Files

- `node-configuration.png` - Screenshot of the node configuration interface in n8n showing all parameters

## Adding the Screenshot

To add the UI screenshot:

1. Take a screenshot of the node configuration panel in n8n
2. Save it as `node-configuration.png` in this directory
3. The image will automatically be displayed in the README.md file

**Recommended screenshot:**
- Shows the Parameters tab with all fields visible
- Includes Download URL, Upload URL, Content Length, HTTP Method, Headers, and Error handling toggle
- Clear and readable at reasonable size (recommended: 1200-1600px width)

## Usage

These images are referenced in the main README.md file to provide visual context for users configuring the node.
